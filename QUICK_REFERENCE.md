# 📖 Dashboard Quick Reference Guide

## 🎯 Main Features at a Glance

### 1. COURSES (វគ្គសិក្សា)
- Browse all available courses
- Filter by category
- Search by title, category, or description
- Track progress with percentage bars
- Save lessons to favorites
- Watch video lessons

**Search Tips:** Type "HTML", "React", "Python", "Web Development", or even Khmer keywords

---

### 2. STATISTICS (ស្ថិតិ)
- 📚 Lessons Completed - Total number
- 📜 Certificates - Exams passed
- ⏱️ Learning Time - Total hours
- ⭐ Favorites - Saved lessons

**Updated in Real-Time** ✨

---

### 3. EXAMS & QUIZZES (ការប្រឡង)
1. Complete all lessons in a course
2. Go to "ការប្រឡង" tab
3. Click "ចាប់ផ្ដើម Quiz"
4. Answer multiple-choice questions
5. Click "ដាក់ស្នើ Quiz"
6. See your score (Green = Correct, Red = Wrong)
7. Click "ទទួលលទ្ធផល" to earn certificate

**Current Quiz Courses:**
- ✅ Web Basic (HTML/CSS/JS)
- ✅ React JS
- ✅ Python Programming

---

### 4. FAVORITES (ចំណូលចិត្ត)
- Click ⭐ to add/remove favorites
- Quick access to saved lessons
- Organized in one place
- Play button to watch

---

### 5. CERTIFICATES (វិញ្ញាបនបត្រ)
- View all earned certificates
- Shows course name and your name
- Beautiful design with award icon
- Appears after passing exam

---

### 6. SETTINGS (ការកំណត់)

#### Profile Picture
- Click camera button
- Choose image file
- Saved in localStorage
- Updates instantly

#### Edit Profile - Add these details:
- **Bio**: Your introduction (Optional)
- **Phone**: Your phone number (Optional)
- **Location**: Where you're from (Optional)
- **Website**: Your personal site (Optional)
- **GitHub**: Your GitHub profile link (Optional)

**Click "កែសម្រួល" → Edit → "រក្សាទុក"**

## 💾 Data Storage (LocalStorage)

### What's Saved:
```
✅ Your profile (bio, phone, location, etc.)
✅ Profile picture (as image data)
✅ Completed lessons
✅ Passed exams
✅ Favorite lessons
✅ Learning statistics
```

### How to Check Saved Data:
1. Open DevTools (Press `F12`)
2. Go to "Application" tab
3. Click "Local Storage"
4. Find your domain
5. Look for: `favorites`, `completedLessons`, `passedExams`, `userProfile`

## ❌ Troubleshooting

### Image Upload Not Working?
- Check file size (less than 5MB)
- Try JPG or PNG format
- Check browser permission
- Clear browser cache

### Quiz Not Showing?
- Complete all lessons first
- Must be 100% progress
- Try refreshing page

### Profile Not Saving?
- Check LocalStorage is enabled
- Not in private/incognito mode
- Click "រក្សាទុក" button clearly

### Data Disappeared?
- Check if you cleared browser data
- Look in LocalStorage (DevTools)
- Data is browser-specific, not cloud-synced
